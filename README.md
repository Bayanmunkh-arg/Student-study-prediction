# Student Score Prediction

## Overview
This project predicts student exam scores based on study hours using machine learning.

## Features
- Predict score from study hours
- Visualize data with graph

## Tech Stack
- Python
- pandas
- scikit-learn
- matplotlib

## Result
The model shows a positive relationship between study hours and exam score.
